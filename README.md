# 🌟 Vacation Fun Worksheet – Class 1

An interactive, colorful learning worksheet for Class 1 students to play and learn during vacations!

## 🎯 Activities Included
- 🍎 **Count the Fruits** — Count emojis and type the answer
- 🔤 **Missing Letters** — Fill in the A–Z alphabet gaps
- ➕ **Fun with Numbers** — Addition & subtraction sums
- 🐾 **Match the Animals** — Match animals to their sounds
- 🔍 **Odd One Out** — Spot what doesn't belong
- 🎨 **Learn Your Colours** — Colour recognition quiz
